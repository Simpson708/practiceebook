# Stock & Crypto Price Tracker (Flask App)

A simple web app that fetches real-time stock and crypto prices using **Yahoo Finance** and displays charts.

## How to Run Locally

1. Install Python 3.10+
2. Run:

```
pip install -r requirements.txt
python app.py
```

3. Open your browser:
```
http://127.0.0.1:5000
```

## Deploy on Replit

1. Upload all files  
2. Replit auto-detects Flask  
3. Click “Run”  
4. You’ll get a public URL
